Part of Hive ransomware v3 source codes to demonstrate encryption algorithm.

The program hive_v3_demo does the following:
- generates key table and exports it to "*.key.whxs7" file in current directory;
- encrypts file "test.dat" in the current directory;
- extracts ransom note "gCPW_HOW_TO_DECRYPT.txt" into the current directory;
- open ransom note.

This program is safe and does not need to be run in VM!
